# Font Licenses

Fonts bundled in this repository are not covered by the top-level MIT license. Each font retains its upstream license. All four fonts allow commercial use and redistribution under the conditions in their individual license files.

| Font | License | Upstream |
|------|---------|----------|
| Inter | SIL Open Font License 1.1 | https://github.com/rsms/inter |
| JetBrains Mono | SIL Open Font License 1.1 | https://github.com/JetBrains/JetBrainsMono |
| Merriweather | SIL Open Font License 1.1 | https://fonts.google.com/specimen/Merriweather |
| IBM Plex Sans | SIL Open Font License 1.1 | https://github.com/IBM/plex |

The full license text for each font lives inside its directory (`fonts/<name>/OFL.txt` or `LICENSE.txt`).

## When Redistributing

If you fork, vendor, or embed these fonts you must:

1. Preserve the per-font license file.
2. Not rename the font Reserved Font Names (e.g. "Inter") unless modifying the font itself. Modifications are allowed but must use a different family name.
3. Not sell the fonts in isolation. Bundling them inside a larger work like this tool is fine.
